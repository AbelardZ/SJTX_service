# 日线复盘

## 执行顺序
数据抓取脚本按以下顺序执行：
1. `data_fetcher_index.py` - 指数数据
2. `data_fetcher_limitup.py` - 涨停数据  
3. `data_fetcher_sw.py` - 申万行业数据
4. `data_fetcher_all_data.py` - 全市场数据
5. `data_fetcher_short.py` - 短线分析数据

使用 `run_all_fetchers.py` 可按顺序自动执行所有脚本。

## 一、指数群

### 数据库
- **数据库**：`DailytChartDB` (MySQL)
- **表命名**：按日期命名: `Index_2025_11_20`
- **字符集**：utf8mb4 (支持中文)

## 二、短线

- **脚本**：`data_fetcher_all_data.py`
- **数据库**：`DailytChartDB` (MySQL)
- **表命名**：按日期命名: `all_data_YYYY_MM_DD`
- **字符集**：utf8mb4 (支持中文)
- **数据来源**：akshare `stock_zh_a_spot()`

本项目是 "A-share auxiliary system" 的一部分，用于每日收盘后抓取和分析涨停股数据。系统自动计算涨停类型（如一字板、秒板等），并提供 Web 界面进行复盘分析和人工标注。

#### 数据库结构
